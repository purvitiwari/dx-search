# Diagnostics Center Search 

## Synopsis
Search Diagnostic centers in a particular area that perform required tests.

## Tech (MERN Stack)
  - ReactJS v15.3
  - Node.js v6.3.1
  - Express v4.14.0
  - MariaDB v10.1.13

## Environment Setup
- Clone Repository
```bash
  git clone https://github.com/purvitiwari/dx-search
```
- Install package manager and dependencies
```bash
	install npm
```
- Go to /dx-search and 
```bash
	  npm install
```
- Run the server
```bash
	  npm start
```
